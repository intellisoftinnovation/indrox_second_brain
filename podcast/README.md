# 🎙️ HiddenLayer — Podcast

Espacio de gestión del podcast **Hidden Layer by Indrox**.

## Estructura

- `invitados/` — Perfiles de invitados (jalados desde Notion)
- `blueprints/` — Blueprints personalizados por episodio (generados por Frieren)

## Cómo funcionar

Cuando necesites un blueprint para un invitado, dile a Frieren:
> "Genera el blueprint para [Nombre]"

Frieren jalará los datos de Notion, elegirá el template correcto (fundador o experto) y creará el blueprint en `blueprints/`.
